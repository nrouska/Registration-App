# My very first project

Flask-based website developed by Natalia Rouska. Includes a form for user registration. Users arestored in .txt file. 

Deployed on Heroku for public access

# Set-up
1. In VScode powershell
2. `python -m venv virtualenv`
3. `.\virtualenv\Scripts\activate.ps1`
4. `pip install Flask`
5. `python wsgi.py`
